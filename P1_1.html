La validacin cruzada es una tcnica de aprendizaje automtico, que se utiliza para evaluar el rendimiento de un modelo dividiendo el conjunto de datos en varias particiones para evaluar el modelo con mltiples combinaciones de datos.
El argumento cv se utiliza para indicar el nmero de particiones en el que se quiere dividir el conjunto de datos.
